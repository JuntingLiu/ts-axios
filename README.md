# Axios TypeScript 版

## 目录结构介绍

```tree
├── CONTRIBUTING.md
├── LICENSE
├── README.md
├── code-of-conduct.md
├── package-lock.json
├── package.json
├── rollup.config.ts  #rollup 配置文件
├── src # 源码目录
├── test # 测试目录
├── tools # 发布相关配置脚本
├── tsconfig.json # 编译配置文件
└── tslint.json # 风格检查配置文件
```

## Feature

---

- [ ] 在浏览器端使用 XMLHttpRequest 对象通讯

- [ ] 支持 Promise API

- [ ] 支持请求和响应的拦截器

- [ ] 支持请求数据和响应数据的转换

- [ ] 支持请求的取消

- [ ] JSON 数据的自动转换

- [ ] 客户端防止 XSRF
